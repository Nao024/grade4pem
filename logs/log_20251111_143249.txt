=== 解析結果 ===
原因
- 計算が終わらず、同じ呼び出しをくり返してしまっています。
  - 例1: gcd(13, 13) は次に gcd(0, 13) になりますが、その次もずっと gcd(0, 13) のままで止まりません。
  - 例2: gcd(3, 12) は 3 < 12 なので 3 % 12 = 3、次も gcd(3, 12) と同じ呼び出しが続きます。
- 理由は、「次の呼び出しに渡す2つの数の順番」が逆だからです。後ろの数が変わらないままなので、いつまでも終わりません。

直し方
- 次の1行の順番を入れ替えます。
  - 間違い: return gcd(a % b, b);
  - 正しく: return gcd(b, a % b);
- ついでに、どちらかがマイナスでも正しく出るように、最初に絶対値にしておくと安心です。

修正後のコード例
public static int gcd(int a, int b) {
    a = Math.abs(a);
    b = Math.abs(b);
    if (b == 0) {
        return a;
    } else {
        return gcd(b, a % b);
    }
}

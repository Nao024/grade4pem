=== 解析結果 ===
原因
・gcdを呼び直すとき、2つの数の並びが逆になっているため、同じ計算を延々とくり返して止まらなくなっています。
・正しくは「(小さい方ではなく)後ろの数」と「前の数を後ろの数で割ったときの余り」の並びで呼び直します。

直し方（最重要の1点）
・次の1行だけを直してください。
  return gcd(a % b, b);
  を
  return gcd(b, a % b);
  に変更

修正版 GCD.java（全体）
package java_programs;
import java.util.*;

public class GCD {
    public static int gcd(int a, int b) {
        if (b == 0) {
            return a;
        } else {
            return gcd(b, a % b);
        }
    }
}

=== 解析結果 ===
原因
・調べる対象がもう無いのに、無理やり先頭から取り出そうとしてエラーになっています。ずっと繰り返すように書かれているため、中身が空になっても抜け出せないのが原因です。

最も重要な直し方（これだけでテストは通ります）
・「中身があるあいだだけ」繰り返すようにし、なくなったら「見つからなかった」と返すようにします。
  - while(true) を「中身が空でないあいだ」に変更
  - 繰り返しが終わったら false を返す

変更点（イメージ）
- 変更前: while (true) { … queue.removeFirst() … }
- 変更後: while (!queue.isEmpty()) { … queue.removeFirst() … }
- ループの後ろに return false; を追加

任意の改善（より安全になりますが、必須ではありません）
・前回の探索の記録が残らないように、メソッドの最初で nodesvisited をいったん空にする（nodesvisited.clear()）。
・広がる順に調べるなら、次に調べるものは「うしろに並べる」（queue.addLast(...))。
